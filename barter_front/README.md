### 说明

这是一个校园二手交易平台前端代码，使用了Vue+ElementUI，包括了前台用户界面和后台数据管理界面。
目前已基本完成，包括了前台用户登录注册、发布商品、二手商品购买，后台管理员日志、分配角色权限等基本功能。
